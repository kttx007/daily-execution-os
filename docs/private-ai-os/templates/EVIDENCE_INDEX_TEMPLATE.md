# Evidence Index — PRODUCT / DOMAIN

## 1. 使用说明

本文件只登记证据，不替代原始文件。每条证据必须有唯一编号，产品事实卡通过编号引用。

证据状态：

- `VERIFIED`：已核对且适用；
- `PENDING`：待核对；
- `CONFLICT`：与其他来源冲突；
- `EXPIRED`：已过期；
- `RETIRED`：已废止；
- `MISSING`：知道应存在但暂未取得。

## 3. 来源优先级

当资料冲突时，默认按以下顺序判断，但仍需负责人确认：

1. 当前实物测试与受控测试记录；
2. 当前版本正式说明书、BOM、研发/供应商确认文件；
3. 当前型号有效认证/测试报告；
4. 已确认的冻结事实卡；
5. 当前包装标签、铭牌、出货清单；
6. 历史规格书与历史订单；
7. 客户聊天、销售口述、旧宣传页；
8. 行业常识或近似型号——不能作为独立证据。

## 6. 大文件登记规则

对于云盘、对象存储中的 PDF、图片和视频，至少记录：

- 文件名与版本；
- 原始链接；
- 上传/提供人；
- 获取日期；
- 文件大小；
- SHA-256（可选但推荐）；
- 访问权限；
- 对应产品与事实字段；
- 是否允许对外使用。
